# Tagger

Tagger is an AI powered web application that allows users to create tags, descriptions and titles for their items.
App allows users to create tags for every product they own.

## Features
- User authentication and profile management
- AI-powered tag, description, and title generation
- Item management with CRUD operations

## Project structure

- `backend/`: Contains the Spring Boot backend code
- `frontend/`: Contains the Angular frontend code (to be implemented)
- `.github/workflows/`: Contains GitHub Actions workflows for CI/CD
